## Dependencies

- [Node](https://nodejs.org) v12.16.3 or higher
- [Yarn](https://yarnpkg.com) v1.22.4 or higher
- Docker
- Docker compose

## Installation

```bash
$ yarn install
```

## Local environment

```bash
$ yarn db:up
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
